<h1 align="center">DOTS Dass 21</h1>

> DOTS 'Dass 21' Form

## Setup

First, install dependencies and copy the .env file:

```bash
npm install & cp .env.example .env.local
```

Then start the app:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
